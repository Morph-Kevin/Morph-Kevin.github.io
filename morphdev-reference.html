API

API - Application Programming Interface
From Wikipedia: An application programming interface is a way for two or more computer programs to communicate with each other. It is a type of software interface, offering a service to other pieces of software.
API is a generic term that encompasses many possible contexts. In the case of Morph Dev an api typically refers to communicating with a sever to get data from or send data to a web based SaaS.

Client

A client is a computer or program that is making requests to a server. Often this refers to the user's computer's web browser when browsing a website.

CSS

CSS - Cascading Style Sheet
Used in combination with HTML CSS adds styling information to your webpage structure. This includes font and element sizing, color information, and even animations.

Database (DB)

A database is a store of information. In the context of a website, a server will query a database for text or images and other content (typically a text link to a file location on a server)

HTML

HTML - HyperText Markup Language
A text encoding system for defining webpage structure. In other words, the framework for how content (text, images, etc) is displayed in the web browser. Right click this page and click inspect to view the html code that is rendering this content.

JavaScript (JS)

JavaScript is an interpreted programming language. Its popularity has led to its use in varied ways accross software development but is most closely associated with internet technologies. Along with HTML and CSS, Javascript is one of the 3 technologies that makes up the modern web (At least for the Client). Where HTML provides the structure of a webpage and CSS provides the visual styling, JavaScript provides the functionality and logic. For example JS can be used to do everything from make network requests to respond to user input on a page. JavaScript is excecuted on the client's computer after the website files have be downloaded from the server.

JSON

JSON - JavaScript Object Notation
JSON is a text based organization of information often used in web technologies. Queries to a Database often return data formatted as JSON
Example:
{
 "exampleKey": "exampleValue",
 "exampleKey2" : [
   "arrayKey": "arrayKeyValue",
   "arrayKey2": "arrayKeyVlaue",
 ]
}

React

React is a javascript framework used for making SPAs. React is one of many javascript frameworks but it is the one most used at Morph Dev

SASS

SaaS - Software as a Service
Software that uses some kind of subscription model for user access e.g. Adobe Creative Suite. This can refer to native downloadable applications as well as cloud based software such as Twilio/SendGrid. In the latter case, often the main feature of the service is its API.

Server

A server is a computer that is listening for network requests and returns a response. The response could be any kind of data but typically it will be a website (HTML/JavaScript/CSS), an image, a video, or a data object of some kind e.g. JSON

SPA

SPA - Single Page Application
An SPA is a webpage where all of the website structure (and sometimes content) is delivered in the initial response from the server. Site navigation takes place via javascript code and content is loaded in dynamically from the server. SPAa have the advantage of a much more responsive user experience since all of the website's code is downloaded to the Client immediately and they tend to feel like a native application rather than a website.
